# Social Media Campaign Performance Analysis using MySQL

## Project Overview
This project focuses on analyzing social media campaign data using MySQL. The objective is to understand campaign performance, user engagement, and key marketing metrics by writing and executing structured SQL queries.

The project simulates a real-world marketing analytics scenario where data is explored and analyzed to support decision-making.

## Tools Used
- MySQL
- SQL

## Dataset Description
The dataset represents social media campaign information, including campaign details, engagement metrics, and performance-related data. It is used to evaluate how different campaigns perform based on available metrics.

## Key Analysis Areas
- Campaign performance comparison  
- User engagement analysis  
- Reach and interaction trends  
- Basic performance metrics evaluation  

## Project Structure
- `social_media_campaign_analysis.sql`  
  Contains all MySQL queries written for data analysis and performance evaluation.

## Key Skills Demonstrated
- Writing optimized SQL queries  
- Filtering and aggregating data  
- Using joins and conditions  
- Analyzing marketing-related datasets  

## How to Use
1. Import the dataset into MySQL.
2. Open the `.sql` file in MySQL Workbench or any MySQL-compatible tool.
3. Execute the queries sequentially to view the results.

## Notes
This project is created for learning and practice purposes to strengthen SQL and data analysis skills using a practical business use case.
